# Keyboard Learning Order: 2 Keys at a Time

A recommended learning order for standard computer keyboard typing, excluding the number keypad.

This order starts with the strongest home-row anchors, then expands outward by finger, then adds numbers, punctuation, and common control keys.

## Cleaner Game-Style Progression

For a typing game, this progression is easier to organize into levels.

### Phase 1: Home Row Control

`F J` → `D K` → `S L` → `A ;` → `G H`

### Phase 2: Top Row Letters

`R U` → `E I` → `W O` → `Q P` → `T Y`

### Phase 3: Bottom Row Letters

`V M` → `C ,` → `X .` → `Z /` → `B N`

### Phase 4: Real Typing Keys

`␣ Enter` → `Shift Backspace` → `Tab Caps Lock`

### Phase 5: Punctuation

`' -` → `[ ]` → `\ =`

### Phase 6: Number Row

`1 0` → `2 9` → `3 8` → `4 7` → `5 6`

### Phase 7: Shifted Symbols

`! )` → `@ (` → `# *` → `$ &` → `% ^` → `: "` → `< >` → `? _` → `{ }` → `| +`

## Notes

For beginners, function keys, arrow keys, Insert, Delete, Home, End, Page Up, and Page Down are usually better taught later as navigation and editing keys, not as core typing keys.
